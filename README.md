# Booking.com

Booking.com is a Spring Boot application for creating the REST Api's for ticket booking.

## Architecture Diagram

![plot](Architecture.png?raw=true "Architecture Diagram")

## Entities

* Booking.java
* Cinema.java
* Cinema_Hall.java
* Cinema_Seat.java
* City.java
* Movie.java
* Payment.java
* Show.java
* Show_Seat.java
* User.java

## Dependancy

* Database - MySql
* Spring boot JPA
* Spring boot Web
